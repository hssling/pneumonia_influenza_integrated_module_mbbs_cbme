## 🚀 **Getting Started**

### **For Medical Students**
```bash
# Clone the diabetes training repository
git clone https://github.com/hssling/diabetes_mellitus_training_material_mbbs_students.git
cd diabetes_mellitus_training_material_mbbs_students

# Install Streamlit for interactive elements
pip install -r requirements.txt

# Run the interactive quiz platform
streamlit run streamlit_app.py --server.port 8501

# Open browser at http://localhost:8501 to access quizzes
```

### **Running Local Web Server**
```bash
# Start Python server for HTML materials
python -m http.server 8000

# Access training materials at http://localhost:8000/
```

## 🎓 **CBME Learning Objectives**

This training platform enables MBBS students to:

1. **Diagnose Diabetes Mellitus** using clinical and laboratory criteria
2. **Manage Type 1 and Type 2 Diabetes** with appropriate treatment regimens
3. **Prevent Acute Complications** (hypoglycemia, DKA, HHS)
4. **Monitor Long-term Complications** (retinopathy, nephropathy, neuropathy)
5. **Educate Patients** on lifestyle modifications and self-management
6. **Provide Community Care** through screening and preventive programs
