## C.1 Interactive Statistical Calculators

### Free Online Calculators
- **VassarStats**: http://vassarstats.net - Comprehensive statistical calculators
- **GraphPad QuickCalcs**: https://www.graphpad.com/quickcalcs - Easy-to-use calculators
- **Social Science Statistics**: https://www.socscistatistics.com - Tests and calculators
- **OpenEpi**: https://www.openepi.com - Epidemiological calculators
- **MedCalc**: https://www.medcalc.org/calc - Medical statistics calculators

### Specialized Calculators
- **Sample Size Calculators**:
  - PS: Power and Sample Size Calculation (biostat.mc.vanderbilt.edu/wiki/Main/PowerSampleSize)
  - G*Power: Free power analysis software (gpower.hhu.de)

- **Diagnostic Test Calculators**:
  - Diagnostic Test Calculator (araw.mede.uic.edu/cgi-bin/testcalc.pl)
  - ROC Curve Analysis (medcalc.org/calc/diagnostic_test.php)

- **Survival Analysis Tools**:
  - Kaplan-Meier Survival Calculator (vassarstats.net/survival.html)
  - Cox Regression Calculator (statskingdom.com/cox-regression-calculator)
