## 📚 Overview

This comprehensive pandemic module has been developed following **Competency-Based Medical Education (CBME)** guidelines as implemented in Indian medical colleges under the National Medical Commission. The module equips future healthcare professionals with the knowledge, skills, and attitudes required for effective pandemic preparedness and management.

### 🎯 Key Features

- **CBME-Aligned Content**: Competency-based curriculum with progressive learning
- **38 Comprehensive Chapters**: From basic concepts to advanced research
- **150+ MCQ Bank**: Competency-based assessment questions
- **Interactive Components**: Streamlit web application for self-learning
- **Real-World Case Studies**: COVID-19 and historical pandemic examples
- **Indian Context**: Specific focus on Indian healthcare systems and policies

## 🎓 Learning Objectives & CBME Competencies

### Knowledge Domain (PA, PB, PC Competencies)
- Explain pandemic epidemiology, clinical management, and public health response
- Describe SARS-CoV-2 virology, COVID-19 spectrum, and vaccination strategies
- Understand ethical, legal, and policy frameworks in pandemic response

### Skills Development
- Perform proper PPE donning/doffing and infection prevention procedures
- Conduct triage and resource allocation in crisis scenarios
- Implement contact tracing and quarantine protocols

### Attitude Formation
- Demonstrate empathy and professionalism in crisis situations
- Value interdisciplinary collaboration in healthcare delivery
- Commit to continuous learning and adaptation in dynamic environments

## 🖥️ Interactive Learning Application

pip install -r requirements.txt

streamlit run streamlit_app.py

## 📊 Assessment Framework

### MCQ Bank Specifications
- **Total Questions**: 150
- **Difficulty Distribution**: 35% Basic, 40% Intermediate, 25% Advanced
- **CBME Alignment**: Mapped to specific competencies
- **Coverage**: All 8 units and 38 chapters
- **Answer Keys**: Provided with detailed explanations

### Performance Standards
- **Excellent (80-100%)**: Comprehensive understanding and application
- **Good (60-79%)**: Satisfactory knowledge with minor gaps
- **Needs Improvement (<60%)**: Requires additional learning support
